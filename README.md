# IoT WeatherDash

IoT WeatherDash is a weather monitoring dashboard that collects environmental data from sensors and displays it through a web-based interface. The project demonstrates how IoT devices can transmit data to a dashboard for real-time monitoring and visualization.



## Project Overview

WeatherDash is designed to monitor environmental conditions such as temperature and humidity using IoT sensors. The collected data is transmitted to a dashboard where users can view updates and track environmental changes.

This project demonstrates concepts such as:

- Internet of Things (IoT) communication
- Sensor data collection
- Real-time data visualization
- Web-based dashboard development



## Features

- Real-time sensor data monitoring
- Temperature and humidity tracking
- Dashboard data visualization
- IoT device communication
- Responsive web interface

## Installation

### Clone the repository

```bash
git clone https://github.com/muhammadsolihinmadsarib/IoT-Project-WeatherDash.git
```

### Navigate to the project folder

```bash
cd IoT-Project-WeatherDash
```

### Install dependencies

```bash
npm install
```

### Run the project

```bash
npm start
```
